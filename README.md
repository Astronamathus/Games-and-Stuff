# Compilation of my playground

This repository just contains a few menial projects I've created, most are simple but some are tedious and are rather archaic. They mostly serve as a testament to my programming ability and depict me as somewhat of a coding aficianado. 

## List of projects - what they do
1) PacMan - Rather obvious, the classic arcade game, but in Java
2) Shapes - Pick your shape and size and let beautifully crafted for loops do the rest
3) Timer - It works, kinda?
4) Wordle - My greatest triumph as of 2025, lots of iterations on this one
5) Wordle_Text - The harbinger of graphical wordle, served as a proof of concept and a test of checking algorithm
6) atomic_number_plot (py) - Saw it in my chem textbook, wanted it on my computer screen
7) brian (js) - Probably one of the first things I coded (ripped it off a youtube video)
8) eggjams - Want to know how badly you failed?
9) nCr - nCu(r)iosity
10) numberguesser (py) - Very original I know, I was getting back to python after like 3 years
11) rps-adaptive (py) - AI-assisted, more like AI-built and I watched
12) rps - Built in school because I was bored
13) snake - Lots of headaches but it works and I'm proud
14) DecimalBin - Encoder that takes decimal input and converts to binary
14) DecimalOct - Encoder that takes decimal input and converts to octal
15) numcompare (C) - First C program, does what it says it does
16) headsortails (py) - Simulates n coin flips and plots the number of heads and tails with % of each
17) Assortment (py) - Randomly chooses one of 11 alphabets repeatedly for chosen frequency, highlighted not so random nature of rng
18) uppercase (C) - Prints inputted word in UPPERCASE
